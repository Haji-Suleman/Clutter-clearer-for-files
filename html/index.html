File Organizer

Automatically organizes your files based on their extensions!

Features

  • Organizes files into directories based on their extensions.
  • Simple web server built with Express.js.
  • Automatically organizes files when the server starts.
  • Basic error handling for file operations.

Installation

  1. Clone the repository:
    2. git clone https://github.com/your-username/file-organizer.git
  2. Navigate to the project directory:
    3. cd file-organizer
  3. Install dependencies:
    4. npm install
Visit on GitHub

Usage

  1. Start the server:
    2. npm start
  2. Open a browser and go to http://localhost:3000 to view the file list.

How It Works

The application reads files in a specified directory and moves them to subdirectories based on their file extensions.

  • Directories are created automatically for each file extension.
  • Files are moved to their respective extension folders.

Error Handling

Errors during file reading or moving are logged to the console for troubleshooting.
